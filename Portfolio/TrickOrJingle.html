Trick or Jingle

Project information

  • Category: Phaser Game
  • Motivation: University project
  • Type: Group project
  • Role: Programmer
  • Project date: January, 2022
  • Technical video: https://youtu.be/e05qWvB7Czk
  • Project URL: https://coexito.itch.io/trick-or-jingle

"Trick or Jingle" is an online multiplayer fighting game, to host the final battle between Halloween and Christmas fans.
The game was made in Phaser with JavaScript, and it was hosted in a Spring server, using Java for the API REST and Spring Boot implementation.
The online multiplayer behaviour can be seen in the linked video, whereas a local multiplayer version can be found in Itch.io.
